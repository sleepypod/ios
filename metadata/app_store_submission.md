## Description

```
sleepypod — local control for your Eight Sleep Pod

Take control of your sleep environment without the cloud. sleepypod connects directly to your Eight Sleep Pod (3, 4, or 5) on your home network, giving you fast, private, and reliable temperature control.

TEMPERATURE CONTROL
- Adjust bed temperature with a precision dial
- Set independent temperatures for each side
- Link both sides for synchronized control

SMART SCHEDULING
- Create temperature curves for the perfect night
- Design custom curves with AI assistance
- Save and reuse your favorite sleep profiles

ON-DEVICE SLEEP ANALYSIS
- Sleep stage classification performed on-device using validated heuristics
- Heart rate and breathing rate extracted from raw piezo signals
- Sleep quality scoring with circadian-aware algorithms
- Track weekly trends and movement patterns

LIVE SENSOR DATA
- Real-time piezo waveform with DSP-based vitals extraction
- Bed temperature matrix with zone-level detail
- Capacitive presence detection with variance analysis
- Data pipeline visualization showing live sensor data flow

HAPTIC ALARMS
- Configure vibration wake-up alarms
- Choose from preset patterns or create custom ones
- Set per-side alarm schedules

PRIVACY FIRST
- Data is processed and stored locally on your pod — no cloud account required
- No telemetry or tracking
- Your sleep data belongs to you

Requires an Eight Sleep Pod (3, 4, or 5) with sleepypod-core installed and running on your home network. Use "Explore Demo" on first launch to preview the full app experience without hardware.
```

## App Review Information

### Notes for Review

```
1. On first launch, tap "Explore Demo" to use the app with simulated data
2. All screens are fully functional in demo mode with realistic sleep data
3. The app controls Eight Sleep Pod mattress covers on local WiFi
4. No account or login required — communicates directly with the device
5. "Connect" uses Bonjour/mDNS to discover devices on the local network
6. Local network permission is required for device discovery and control

DEMO MODE provides simulated data including:
- Device status (temperature, power state)
- 7 nights of sleep records with realistic vitals
- Live sensor waveforms and pipeline visualization
- Schedule management and settings

To exit demo: Settings gear (top-right) → "Connect to Real Pod"
```
